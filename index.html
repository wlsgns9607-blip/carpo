💡 조진훈의 기획 및 개발 이야기

"자동차는 단순한 이동 수단이 아닙니다."
가족, 연인, 친구들과 함께 소중한 추억을 쌓아가는 공간이자,
여행과 드라이브의 설렘을 전하는 매개체입니다.

고속도로에서의 갑작스러운 차량 고장은 구동벨트, 점화플러그 같은 핵심 부품들의 방치에서 시작됩니다. 평소 잘 모른다는 이유로 무심히 지나쳤던 작은 신호들이, 때로는 생명을 위협하는 위험한 2차 사고로 이어지기도 합니다.

사람의 생명은 그 무엇과도 바꿀 수 없기에,
조기 점검과 예방 교체로 2차 사고를 방지하고자 '카닥터'를 만들었습니다.

Problem Statement

왜 카닥터가 필요한가

운전자들이 정비소에서 겪는 두 가지 핵심 문제

01
🔇 증상을 설명할 수가 없다
차에서 이상한 소리·냄새가 나도 정확한 원인을 몰라 불안합니다. 주행 중 스마트폰 검색은 위험하고, 텍스트만으로는 차량 소음을 설명하기 어렵습니다.
02
💸 바가지를 맞는다
자동차 지식이 없으면 정비소에서 불필요한 수리를 권유받아도 거절하기 어렵습니다. 공식센터 vs 공임나라 가격 차이가 최대 3~4배에 달합니다.
03
⏰ 핵심 부품을 방치한다
엔진오일·타이어는 알아도, 타이밍벨트·휠 얼라이먼트는 비용 부담으로 미룹니다. 방치하면 20~30만원 추가 비용 또는 사고로 이어집니다.
Core Features

6가지 핵심 기능

차량 관리의 모든 것을 하나의 앱으로

🤖
AI 증상 진단
소리·냄새·느낌으로 증상을 입력하면 AI가 즉시 원인을 분석합니다. 픽토그램 선택 또는 텍스트 입력 모두 가능합니다.
키워드 매핑 → 추후 AI API 연동
🛡️
바가지 방지 카드
AI 진단 결과를 카드 형태로 자동 생성합니다. 정비소에서 사장님께 직접 보여주면 됩니다. 예상 수리비도 함께 표시됩니다.
진단 결과 자동 생성
📊
10만km 부품 대시보드
현재 주행거리 기준으로 핵심 부품 교체 시기를 게이지로 시각화합니다. 80% 이상이면 교체 권장 알림이 뜹니다.
주행거리 기준 % 계산
💰
수리비 비교 계산기
공식 서비스센터 vs 공임나라 비용을 항목별로 실시간 비교합니다. 체크박스 선택만으로 얼마를 절약할 수 있는지 바로 확인합니다.
실시간 합산 계산
🗺️
정비소 길안내 연동
AI 진단 후 바가지방지카드 하단에서 카카오맵·네이버지도로 가까운 정비소를 바로 찾을 수 있습니다.
카카오맵 / 네이버지도 딥링크
🚨
불법 렉카 신고
원터치 신고서 자동 생성, 영상·음성 녹화(MediaRecorder API), 내 권리 카드, 신고 기관 안내(112, 1372, 1588-2001)까지 한 번에.
MediaRecorder API 실제 연동
Parts Management

사람들이 가장 미루는 핵심 부품

엔진오일·타이어 말고, 진짜 돈 나가는 부품들

부품 교체 주기 방치 시 위험 상태
⚡ 점화플러그 40,000km 연비 저하, 시동 불량
🔄 휠 얼라이먼트 (토우/캠버만 조정 가능) 20,000km 타이어 편마모 → 사고, 20~30만원 추가
💧 냉각수 40,000km 엔진 과열 → 엔진 손상
🛑 브레이크 디스크 70,000km 제동 불량 → 사고
⚙️ 구동벨트 80,000km 시동 불가
🔧 타이밍벨트 100,000km 끊기면 엔진 전손 위험

※ 위 게이지는 제네시스 G80 / 120,000km 기준 데모 데이터입니다
